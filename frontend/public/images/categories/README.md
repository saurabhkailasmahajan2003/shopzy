# Category Images

Place your PNG images for categories in this folder:

- `men.png` - Image for Men's category (80x80px recommended)
- `women.png` - Image for Women's category (80x80px recommended)
- `watches.png` - Image for Watches category (80x80px recommended)
- `lenses.png` - Image for Lenses category (80x80px recommended)

The images should be square PNG files with transparent backgrounds for best results.

If images are not found, the app will fallback to icon images from Flaticon.
